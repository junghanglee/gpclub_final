GPCLUB 관리자 사용설명서 - 초보 운영자용

시스템을 잘 모르는 운영자도 따라 할 수 있는 GPCLUB Vietnam 웹사이트 운영 가이드
Updated: 2026-06-08PDF Download VersionFor non-technical operators
이 문서는 운영자가 직접 로그인하고, 내용을 수정하고, 저장한 뒤 공개 페이지에서 확인할 수 있도록 쉽게 작성한 안내서입니다.

이 문서는 시스템이나 개발을 잘 모르는 운영자가 그대로 따라 할 수 있도록 만든 설명서입니다. 코딩 지식은 필요 없습니다.

  • 관리자 로그인 페이지: /auth
  • 관리자 페이지: /admin
  • 저장 후 확인할 공개 페이지: /, /products, /catalog, /events, /contact

1. 가장 중요한 기본 원칙

1. 관리자 화면에서 내용을 수정합니다.

2. Save 또는 저장 버튼을 누릅니다.

3. 성공 메시지가 나올 때까지 기다립니다.

4. 실제 공개 페이지를 열어 바뀐 내용이 보이는지 확인합니다.

5. 가능하면 휴대폰 화면이나 작은 화면에서도 한 번 확인합니다.

Save 버튼을 누른 직후 바로 창을 닫지 마세요. 반드시 저장 성공 메시지를 확인한 뒤 이동하거나 닫는 것이 안전합니다.

2. 로그인 방법

1. /auth 페이지를 엽니다.

2. 관리자 이메일과 비밀번호를 입력합니다.

3. 로그인 후 /admin 페이지를 엽니다.

4. 권한이 없다는 메시지가 나오면, 계정에 admin 권한이 없는 상태입니다. 시스템 담당자에게 권한 부여를 요청하세요.

5. 작업이 끝나면 Sign out 버튼으로 로그아웃합니다.

관리자 비밀번호는 채팅방, 이메일 단체방, 메신저에 공유하지 마세요.

3. 관리자 메뉴 쉽게 이해하기

Dashboard는 전체 현황을 보는 곳입니다.

Product Management는 상품과 카탈로그를 관리하는 곳입니다.

Customer Management는 대리점 신청과 고객 문의를 확인하는 곳입니다.

Content Management는 홈페이지, 이벤트, 팝업, FAQ, 챗봇 내용을 수정하는 곳입니다.

Settings는 회사 연락처 정보를 수정하는 곳입니다.

어디로 들어가야 할지 모르겠다면 먼저 Dashboard를 보고, 관련 메뉴로 이동하면 됩니다.

4. Dashboard: 매일 확인할 것

Dashboard에서는 새로 들어온 내용을 빠르게 확인합니다.

확인할 항목:

  • 새 대리점 신청이 있는지
  • 새 문의 메시지가 있는지
  • 진행 중인 팝업이나 이벤트가 있는지
  • 챗봇 데이터 업데이트가 필요한지

새 숫자가 보이면 Customer Management로 이동해서 처리합니다.

5. Products: 상품 추가 또는 수정

상품을 새로 등록하거나 기존 상품 정보를 수정할 때 사용합니다.

상품 추가 순서:

1. Product Management > Products로 이동합니다.

2. New Product 버튼을 누릅니다.

3. Brand Name, Product Name, Product Type을 입력합니다.

4. Short Intro에는 짧고 이해하기 쉬운 설명을 입력합니다.

5. Cover Image에는 공개적으로 열리는 이미지 URL을 입력합니다.

6. 필요하면 상세 설명, 장점, 사용법을 입력합니다.

7. 공개할 준비가 끝나면 Published를 켭니다.

8. Save를 누릅니다.

9. /products 페이지를 열어 상품이 제대로 보이는지 확인합니다.

자주 쓰는 스위치:

  • Published: 켜면 상품이 웹사이트에 공개됩니다.
  • New Product: 신상품 표시를 붙입니다.
  • Popular Product: 인기 상품으로 표시합니다.
  • Featured Product: 주요 영역에서 더 눈에 띄게 표시합니다.

Published를 켜기 전에는 상품명, 이미지, 브랜드명, 설명, 모바일 화면을 꼭 확인하세요.

6. Catalog Management: 파트너용 카탈로그 만들기

카탈로그는 B2B 파트너에게 보여주거나 전달할 상품 목록 자료입니다.

카탈로그 생성 순서:

1. Product Management > Catalog Management로 이동합니다.

2. New Catalog 버튼을 누릅니다.

3. 제목, 부제목, 설명을 입력합니다.

4. 템플릿을 선택합니다. 잘 모르겠으면 Premium을 선택하면 됩니다.

5. 카탈로그에 넣을 상품을 선택합니다.

6. 현재 대표 카탈로그로 사용할 자료라면 Representative Catalog를 켭니다.

7. Save를 누릅니다.

8. /catalog 페이지를 열어 확인합니다.

대표 카탈로그는 현재 공식적으로 보여줄 메인 카탈로그입니다. 한 번에 하나만 대표로 사용하는 것이 좋습니다.

7. Dealers: 대리점 신청 처리

대리점 또는 파트너 신청을 확인하고 처리하는 메뉴입니다.

처리 순서:

1. Customer Management > Dealers로 이동합니다.

2. 새 신청서를 엽니다.

3. 회사명, 담당자, 이메일, 전화번호, 도시, 예상 물량, 메시지를 확인합니다.

4. 필요하면 신청자에게 연락합니다.

5. 내부 메모를 남깁니다.

6. 처리 상태를 변경합니다.

7. 저장합니다.

추천 상태값:

  • New: 새로 들어온 신청
  • Reviewing 또는 In Progress: 검토 중
  • Contacted: 연락 완료
  • Approved: 협력 가능성이 있어 승인
  • Rejected 또는 Closed: 진행하지 않거나 처리 완료

스팸이나 중복 데이터가 아니라면 신청서를 삭제하지 않는 것이 좋습니다.

8. Contacts: 고객 문의 처리

고객 문의나 챗봇/문의 기록을 확인하는 메뉴입니다.

처리 순서:

1. Customer Management > Contacts로 이동합니다.

2. 새 메시지를 먼저 확인합니다.

3. 처리 후 상태를 업데이트합니다.

4. 중요한 기록은 남겨둡니다.

5. 스팸이나 잘못된 중복 데이터만 삭제합니다.

고객 정보는 개인정보입니다. 권한이 있는 담당자 외에는 공유하지 마세요.

9. Home: 홈페이지 수정

홈페이지의 첫 화면, 문구, 이미지, 숫자, 버튼 내용을 수정할 때 사용합니다.

수정 순서:

1. Content Management > Home으로 이동합니다.

2. Page to Edit에서 Home을 선택합니다.

3. 제목, 설명, 버튼, 이미지 내용을 수정합니다.

4. Save Home을 누릅니다.

5. / 페이지를 열어 홈페이지를 확인합니다.

자주 보이는 입력 항목:

  • Kicker: 큰 제목 위에 들어가는 짧은 문구
  • Title: 메인 제목
  • Subtitle: 제목 아래 설명 문구
  • Primary CTA: 주요 버튼 문구
  • Secondary CTA: 보조 버튼 문구
  • Image URL: 이미지 주소
  • Image Alt: 이미지 설명

Reset 버튼은 기본 내용으로 되돌릴 때 사용합니다. 잘 모르겠으면 누르지 않는 것이 안전합니다.

10. Events: 이벤트 또는 신상품 소식 등록

이벤트, 신상품 출시, 주요 공지를 올릴 때 사용합니다.

등록 순서:

1. Content Management > Events로 이동합니다.

2. New Event 버튼을 누릅니다.

3. Event 또는 New Product Spotlight를 선택합니다.

4. 베트남어 제목과 영어 제목을 입력합니다.

5. 짧은 설명을 입력합니다.

6. 필요하면 이미지나 영상 URL을 추가합니다.

7. 중요하게 보여줄 내용이면 Featured를 켭니다.

8. 공개 준비가 끝나면 Published를 켭니다.

9. Save를 누릅니다.

10. /events 페이지를 열어 확인합니다.

두 언어의 내용이 모두 확인된 후에 Published를 켜세요.

11. Popups: 웹사이트 알림 띄우기

팝업은 중요한 공지, 캠페인, 프로모션을 방문자에게 보여줄 때 사용합니다.

생성 순서:

1. Content Management > Popups로 이동합니다.

2. New Popup 버튼을 누릅니다.

3. 제목과 내용을 입력합니다.

4. 필요하면 이미지를 추가합니다.

5. 다른 페이지로 이동시키고 싶으면 CTA 버튼 정보를 입력합니다.

6. 예약 노출이 필요하면 시작 시간과 종료 시간을 설정합니다.

7. Active를 켭니다.

8. Save를 누릅니다.

9. 실제 웹사이트에서 팝업이 보이는지 확인합니다.

팝업이 보이지 않으면 Active, 시작 시간, 종료 시간을 먼저 확인하세요.

12. FAQs: 자주 묻는 질문 추가

고객이 자주 묻는 질문과 답변을 추가할 때 사용합니다.

추가 순서:

1. Content Management > FAQs로 이동합니다.

2. 수정할 언어를 선택합니다.

3. New FAQ 버튼을 누릅니다.

4. 질문을 입력합니다.

5. 답변을 입력합니다.

6. 카테고리를 선택하거나 입력합니다.

7. Published를 켭니다.

8. Save를 누릅니다.

답변은 짧고 명확하게 작성하는 것이 좋습니다.

13. Chatbot: 챗봇 답변 업데이트

챗봇은 크게 네 종류의 데이터를 사용합니다.

  • Q&A: 짧은 질문과 답변
  • Product Info: 상품 정보
  • Document Library: 챗봇이 참고할 긴 문서
  • Tree Scenario: 버튼을 눌러 이동하는 챗봇 흐름

Q&A 추가

1. Content Management > Chatbot으로 이동합니다.

2. Q&A 영역을 엽니다.

3. New Training Entry를 누릅니다.

4. 질문과 답변을 입력합니다.

5. Enabled를 켭니다.

6. Save를 누릅니다.

상품 정보 추가

1. Product Info를 엽니다.

2. 상품명 또는 제목을 입력합니다.

3. 장점, 사용법, 추천 포인트를 입력합니다.

4. Enabled를 켭니다.

5. Save를 누릅니다.

긴 문서 추가

1. Document Library를 엽니다.

2. New Document를 누릅니다.

3. Title과 Raw Content를 입력합니다.

4. language와 category를 선택합니다.

5. Save를 누릅니다.

6. 시스템이 문서를 작은 조각으로 처리할 때까지 기다립니다.

챗봇 답변이 바로 바뀌지 않으면 Enabled가 켜져 있는지, 문서 처리가 완료됐는지 확인하세요.

14. Settings: 회사 연락처 수정

회사 이메일, 전화번호, 주소, Zalo, WhatsApp, 문의 링크를 수정할 때 사용합니다.

수정 순서:

1. Settings로 이동합니다.

2. 연락처 정보를 수정합니다.

3. Save Changes를 누릅니다.

4. /contact 페이지와 웹사이트 하단 푸터를 확인합니다.

공개해도 되는 연락처만 입력하세요.

15. 저장 후 확인 체크리스트

내용을 수정한 뒤에는 아래를 확인하세요.

  • Save 버튼을 눌렀는지
  • 성공 메시지가 나왔는지
  • 공개 페이지에 새 내용이 보이는지
  • 모바일 화면이 깨지지 않는지
  • 영어와 베트남어 의미가 같은지
  • 이미지가 정상 표시되는지
  • 버튼 링크가 올바른 페이지로 이동하는지

16. 자주 생기는 문제 해결

로그인이 안 될 때:

  • 이메일과 비밀번호를 다시 확인합니다.
  • /auth 페이지에서 다시 시도합니다.
  • 계속 안 되면 시스템 담당자에게 문의합니다.

저장했는데 웹사이트가 안 바뀔 때:

  • 공개 페이지를 새로고침합니다.
  • Published 또는 Active가 켜져 있는지 확인합니다.
  • 몇 초 기다린 뒤 다시 확인합니다.

이미지가 안 보일 때:

  • 이미지 URL을 브라우저에서 직접 열어봅니다.
  • URL이 열리지 않으면 다른 이미지 주소로 교체해야 합니다.

팝업이 안 보일 때:

  • Active가 켜져 있는지 확인합니다.
  • 시작 시간과 종료 시간을 확인합니다.
  • 우선순위가 더 높은 다른 팝업이 있는지 확인합니다.

챗봇 답변이 맞지 않을 때:

  • 데이터가 Enabled 상태인지 확인합니다.
  • 문서라면 처리가 완료됐는지 확인합니다.
  • Q&A를 더 짧고 명확하게 다시 작성해봅니다.

17. 안전 규칙

  • 관리자 비밀번호를 공유하지 마세요.
  • 확실하지 않으면 데이터를 삭제하지 마세요.
  • 내부 정보는 공개 입력칸에 넣지 마세요.
  • 승인되지 않은 가격, 인증, 효과 보장 문구를 올리지 마세요.
  • 고객 개인정보를 권한 없는 사람에게 공유하지 마세요.
  • 중요한 수정 후에는 공개 페이지 확인 화면을 캡처해두면 좋습니다.
